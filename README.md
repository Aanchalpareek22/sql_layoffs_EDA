# SQL Layoffs Data Analysis (EDA)

This project is an Exploratory Data Analysis (EDA) performed using **SQL** on a global layoffs dataset.

I worked on this project while learning SQL by following a YouTube tutorial series.  
The main goal was to practice **real-world SQL queries**, understand how data cleaning works, and explore trends in layoffs across different companies, industries, and years.

## What This Project Covers

- Creating staging tables for safe data cleaning  
- Handling NULL values and inconsistent data  
- Exploring layoffs based on:
  - Companies
  - Industries
  - Countries
  - Total layoffs and percentage layoffs
- Finding maximum, minimum, and trend-based insights using SQL queries

## Tools & Technologies

- MySQL  
- SQL (Joins, Aggregations, Filtering, EDA queries)
